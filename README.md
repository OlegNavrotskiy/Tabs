Для правильной работы tab нужно прописать путь

до таких основных переменных:

tab = document.querySelectorAll('.info-header-tab') - все табы

info = document.querySelector('.info-header') - родитель табов (блок со всеми табами)

tabContent = document.querySelectorAll('.info-tabcontent') - вся статьи с информацией до каждого таба
